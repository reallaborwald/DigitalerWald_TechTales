# Digitaler Wald Tech Tales





### How to set up

1. Download Processing.org and the Arduini IDE on the Rasberry Pi
2. Connect the Potentiometers to the ESP32
3. Connect the ESP32 to the Rasberry Pi via USB Cable
4. Open the Arduino IDE and run the Arduino Code
5. Open Processing and paste the .pde file
6. Run the Processing code

Wiring:
| Poteniometer  | ESP32 |
| ------------- | ------------- |
| VCC           | 3.3V  |
| GND           | GND  |
| Output        | GPIO 36, 39, 34, 35 |
